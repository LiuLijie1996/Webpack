### 安装

```
npm install url-loader --save-dev
	这个loader可以把图片解析成base64位格式的字符串
	解析成base64格式字符串可以减少网络请求,只针对小图片

npm install file-loader --save-dev
	第三方的file-loader作用是,在出口处生成一个图片,该图片的名字hash.ext组成,并且返回hash.ext组成的字符串
```



### webpack.config.js配置

```js
//处理非js之外的文件
module: {
    //配置规则
    rules: [
        {
            test: /\.(jpg|png|gif)$/,//匹配到gif结尾的文件
            use: [
                {
                    //注册插件
                    loader: 'file-loader',
                    //插件的配置
                    options: {
                        /*
                        * 单位是字节,小于8K将图片解析成base64,大于8K则处理成图片
                        * 需要安装 file-loader 插件,安装即可,不用注册
                        * */
                        limit: 8192,
                        /*
                        * [name]  使用原始的文件名
                        * [ext]  使用原始的后缀
                        * [path]  资源相对于 context的路径
                        * [hash]  内容的哈希值; [hash:8] 限制长度只要8位
                        * [N]  当前文件名按照查询参数 regExp 匹配后获得到第 N 个匹配结果
                        * */
                        name:'./imgs/[name].[ext]',//保存目录和命名
                    }
                }
            ],
        }
    ]
}
/*
	file-loader 使用解析：
		https://www.webpackjs.com/loaders/file-loader/#%E7%94%A8%E6%B3%95
	
	url-loader 使用解析：
		https://www.webpackjs.com/loaders/url-loader/
*/
```
