### 安装

```
npm install style-loader -D
	生成style标签，并且把style标签插入到head标签中

npm install css-loader -D
	处理css文件并解析css代码
	
npm install less less-loader -D
	安装less表示可以写less代码
	less-loader处理less代码
```



### webpack.config.js配置

```js
module: {
    rules: [
        {
            test: /\.(css|less$)$/,//匹配到css或less结尾的文件
            //如果插件不需要配置,可以用这种写法
            use: ['style-loader', 'css-loader', 'less-loader'],
        },
    ]
}
```
