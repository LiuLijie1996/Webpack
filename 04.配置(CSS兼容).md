## 安装

```html
npm install postcss-loader autoprefixer --save-dev
	处理css的兼容前缀，必须搭配autoprefixer
	
浏览器版本：https://github.com/browserslist/browserslist#readme
```



## 根目录中新建postcss-config.js文件

```js
module.exports = {
    plugins: [
        require("autoprefixer")
    ],
};
```



## webpack.config.js配置

```js
module: {
    rules: [
        {
            test: /\.(css|scss)$/,//匹配到css或scss结尾的文件
            //如果插件不需要配置,可以用这种写法
            use: [
                'style-loader',
                {
                    loader: 'css-loader',
                    options: {
                        importLoaders: 2,//下面有多少loader就写几
                    }
                },
                'sass-loader',// 一个loader
                'postcss-loader',//两个loader
            ]
        },
    ]
}
```



## 配置package.json

```json
{
  //...
  "browserslist":[
    "ie 6-8",
    "> 0.5",
```
