### 安装

```
npm install postcss-loader -D
	处理css的兼容前缀，必须搭配autoprefixer

npm install autoprefixer -D
```



### 根目录中新建postcss-config.js文件

```js
module.exports = {
    plugins: [
        require("autoprefixer")
    ]
};
```



### webpack.config.js配置

```js
module: {
    rules: [
        {
            test: /\.css$/,//匹配到css结尾的文件
            //如果插件不需要配置,可以用这种写法
            use: ['style-loader', 'css-loader', 'postcss-loader'],
        },
    ]
}
```
