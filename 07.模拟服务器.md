### 安装

```
npm install webpack-dev-server --save-dev
	配置本地服务器，npx webpack-dev-server 启动服务
```



### webpack.config.js配置

```js
//服务器配置
devServer: {
    //是否自动打开浏览器
    open:false,
        
	//设置服务端口
    port:3000,
        
	//自动访问该目录下的资源
    contentBase:"./dist",
        
    //代理; 配置跨域问题
    proxy:{
        '/api':{
            //检测到请求的路由是/api开头,就转向target对应的路由地址
            target:'http://localhost:8080',
		   //路径重写
            pathRewrite:{'/api': ''},
        }
    }
},
```



### package.json配置

```json
"scripts": {
    "build": "webpack",//现在可以通过 npm run build 进行打包了
    "dev": "webpack-dev-server"//现在可以通过 npm run dev 启动服务了
},
```
